### What it will be doing?
#### It will be generating random titles which are funny for news

### Concepts Used

- list -> To store group of words like names, actions and places
- random.choice() -> To pick one item random from a list
- print() -> To show the fake headline
- input() -> To ask user if they want another headline
- string concatenation or f-string - To form a sentence using selected words 

### Flow

- Step 1 - import the random module
- Step 2 - Create lists of subjects, actions. and places - all with indian examples
- Step 3 - use random.choice() to pick one word from each list 
- Step 4 - Combine the words into a headline using string formatting
- Step 5 - Print the headline to the user
- Step 6 - Ask the user if they want another headline
- Step 7 - If yes, repeat; if no, stop the program.
